# Simple Calculator (C)

A simple menu-driven calculator program written in C that performs basic arithmetic operations such as addition, subtraction, multiplication, and division using switch-case.

## Features
- Menu-driven interface
- Addition, subtraction, multiplication, and division
- Uses switch-case statement
- Handles division by zero
- Beginner-friendly and easy to understand

## Technologies Used
- C Programming Language
- Standard I/O library (`stdio.h`)
